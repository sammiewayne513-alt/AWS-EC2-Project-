# AWS VPC & EC2 Deployment Project

## Overview
This project demonstrates how to design and deploy a secure AWS environment using a custom Virtual Private Cloud (VPC), public subnet, security groups, and an EC2 instance.

## Skills Demonstrated
- AWS VPC configuration
- Subnets and route tables
- Internet Gateway setup
- Security Groups
- EC2 instance deployment
- Basic cloud networking concepts

## Architecture
- Custom VPC
- Public subnet
- Internet Gateway
- Route table
- Security group allowing SSH/HTTP
- EC2 instance (Amazon Linux)

## Steps Performed
1. Created a custom VPC
2. Created a public subnet
3. Attached an Internet Gateway
4. Configured route tables
5. Created a security group
6. Launched an EC2 instance
7. Verified connectivity

## Tools Used
- AWS Management Console
- EC2
- VPC
- GitHub

## Outcome
Successfully launched and accessed an EC2 instance in a custom VPC, demonstrating foundational cloud infrastructure skills.
